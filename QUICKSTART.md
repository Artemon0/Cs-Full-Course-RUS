## Первое упражнение

Попробуйте прямо сейчас!

```csharp
// Создайте Variables для вашего игрового персонажа
string heroName = "Ваше имя";
int heroLevel = 1;
int heroHealth = 100;
int heroMana = 50;

// Выведите информацию
Console.WriteLine($"Герой: {heroName}");
Console.WriteLine($"Level: {heroLevel}");
Console.WriteLine($"HP: {heroHealth}");
Console.WriteLine($"Мана: {heroMana}");

// Персонаж получил урон
heroHealth -= 30;
Console.WriteLine($"После боя HP: {heroHealth}");

// Персонаж использовал заклинание
heroMana -= 20;
Console.WriteLine($"После заклинания Мана: {heroMana}");
```

Скопируйте этот код в метод Main() любого файла и запустите!

## Problems?

### Программа не запускается

1. Проверьте, что файл сохранен (Ctrl+S)
2. Проверьте, нет ли ошибок (красные подчеркивания)
3. Убедитесь, что метод Main() существует

### Errors компиляции

1. Читайте сообщение об ошибке
2. Проверьте точки с запятой ;
3. Проверьте скобки { }
4. Используйте Alt+Enter для Quick Fix

### Не понимаю код

1. Читайте Comments
2. Запускайте по частям
3. Меняйте значения и смотрите результат
4. Задайте вопрос в [FAQ](FAQ.md)
