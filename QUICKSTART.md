## 5 Minutes to Your First Lesson

### Step 1: Install Rider or Visual Studio (if not already done)

**Rider (recommended):**

- Download: https://www.jetbrains.com/rider/
- Free for students

**Visual Studio Community (free):**

- Download: https://visualstudio.microsoft.com/

### Step 2: Open the Project

1. Open Rider/Visual Studio
2. File → Open → select the `MyLearn` folder
3. Wait for the project to load

### Step 3: Run Your First Example

1. Open `MyLearn/Module01_CSharpBasics/01_Variables.cs`
2. Press F5 (or click Run)
3. See the result in the console!

### Step 4: Experiment

Change the code:

```csharp
int playerHealth = 100;  // Change to 50
```

Run again (F5) and see what changed!

### Step 5: Complete Exercises

Each file has exercises at the end. Try to complete them!

## Первое упражнение

Попробуйте прямо сейчас!

```csharp
// Создайте Variables для вашего игрового персонажа
string heroName = "Ваше имя";
int heroLevel = 1;
int heroHealth = 100;
int heroMana = 50;

// Выведите информацию
Console.WriteLine($"Герой: {heroName}");
Console.WriteLine($"Level: {heroLevel}");
Console.WriteLine($"HP: {heroHealth}");
Console.WriteLine($"Мана: {heroMana}");

// Персонаж получил урон
heroHealth -= 30;
Console.WriteLine($"После боя HP: {heroHealth}");

// Персонаж использовал заклинание
heroMana -= 20;
Console.WriteLine($"После заклинания Мана: {heroMana}");
```

Скопируйте этот код в метод Main() любого файла и запустите!

## Problems?

### Программа не запускается

1. Проверьте, что файл сохранен (Ctrl+S)
2. Проверьте, нет ли ошибок (красные подчеркивания)
3. Убедитесь, что метод Main() существует

### Errors компиляции

1. Читайте сообщение об ошибке
2. Проверьте точки с запятой ;
3. Проверьте скобки { }
4. Используйте Alt+Enter для Quick Fix

### Не понимаю код

1. Читайте Comments
2. Запускайте по частям
3. Меняйте значения и смотрите результат
4. Задайте вопрос в [FAQ](FAQ.md)
