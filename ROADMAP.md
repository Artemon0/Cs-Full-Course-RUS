## Контрольные точки

### ✅ Контрольная точка 1: После Модуля 1
**Вы должны уметь:**
- Создавать переменные всех типов
- Использовать операторы
- Писать условия и циклы
- Работать с коллекциями
- Создавать методы

**Тест:** Создайте простую консольную игру "Угадай число"

### ✅ Контрольная точка 2: После Модуля 2
**Вы должны уметь:**
- Создавать классы
- Использовать наследование
- Работать с интерфейсами
- Применять паттерны

**Тест:** Создайте систему персонажей с наследованием

### ✅ Контрольная точка 3: После Модуля 4
**Вы должны уметь:**
- Работать с Unity
- Создавать игровые механики
- Использовать физику
- Создавать UI

**Тест:** Создайте простую игру с движением и стрельбой

### ✅ Контрольная точка 4: После Модуля 7
**Вы должны уметь:**
- Создавать полноценные игры
- Применять все изученное
- Оптимизировать код
- Отлаживать проблемы

**Тест:** Создайте свою уникальную игру
