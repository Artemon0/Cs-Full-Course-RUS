# 🎮 C# для начинающих разработчиков игр

Интерактивный курс изучения C# с игровыми примерами. Все уроки можно запускать и экспериментировать с кодом.

## 📚 Содержание курса

### Модуль 1: C# Basics (6 уроков)
- ✅ **Lesson 1**: Variables и типы данных
- ✅ **Lesson 2**: Operators (арифметические, логические, сравнения)
- ✅ **Lesson 3**: Conditional Statements (if, switch)
- ✅ **Lesson 4**: Loops (for, while, foreach)
- ✅ **Lesson 5**: Arrays and Collections (List, Dictionary)
- ✅ **Lesson 6**: Methods и Functions

### Module 2: OOP in C# (4 урока)
- ✅ **Lesson 7**: Classes and Objects
- ✅ **Lesson 8**: Inheritance and Polymorphism
- ✅ **Lesson 9**: Interfaces and Abstract Classes
- ✅ **Lesson 10**: Design Patterns (Singleton, Factory, Observer)

### Модуль 3: Игровые концепции (3 урока)
- ✅ **Lesson 11**: Игровой цикл (Update, deltaTime)
- ✅ **Lesson 12**: Компонентная система (GameObject, Component)
- ✅ **Lesson 13**: Система событий (Events, delegates)

## 🚀 Как запустить

### Вариант 1: Через командную linesу
```bash
cd MyLearn
dotnet run
```

### Вариант 2: В JetBrains Rider
1. Откройте проект `MyLearn.csproj` в Rider
2. Откройте файл `Program.cs`
3. Нажмите зеленую кнопку ▶️ Run
4. Выберите урок из меню (1-10)

### Вариант 3: В Visual Studio
1. Откройте проект `MyLearn.csproj`
2. Нажмите F5 или Ctrl+F5
3. Выберите урок из меню

## 📖 Как учиться

1. **Запустите урок** - выберите номер из меню
2. **Читайте код** - все примеры с подробными комментариями
3. **Экспериментируйте** - меняйте код и смотрите результат
4. **Делайте упражнения** - в конце каждого урока есть задания
