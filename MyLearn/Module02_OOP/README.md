# Модуль 2: Object-Oriented Programming (ООП)

## 🎯 Цели модуля

После прохождения этого модуля вы будете:
- Понимать принципы ООП
- Создавать classes и объекты
- Использовать inheritance и полиморфизм
- Работать с интерфейсами
- Применять patterns проектирования

## 📚 Содержание

1. **[01_Classes.cs](01_Classes.cs)** - Classes and Objects
2. **[02_Inheritance.cs](02_Inheritance.cs)** - Inheritance and Polymorphism
3. **[03_Interfaces.cs](03_Interfaces.cs)** - Interfaces and Abstract Classes
4. **[04_Patterns.cs](04_Patterns.cs)** - Design Patterns

## 🎮 Практические примеры

- Система персонажей (Player, Enemy, NPC)
- Hierarchy оружия (Weapon -> Sword, Bow, Staff)
- Система врагов с inheritanceм
- Паттерны: Singleton, Factory, Observer

Рекомендуемое время: **3-4 недели**
