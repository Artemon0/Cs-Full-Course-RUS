# Модуль 3: Игровые концепции на C#

Этот модуль показывает игровые концепции без Unity - чистый C# код, который потом легко перенести в Unity.

## Lessonи:

1. **Игровой цикл** - Update, deltaTime, игровые объекты
2. **Компонентная система** - GameObject, Component pattern
3. **Система событий** - Events, delegates для игровых систем

Все примеры работают в консоли и демонстрируют принципы, которые используются в Unity.
