# Модуль 1: C# Basics

## 🎯 Цели модуля

После прохождения этого модуля вы будете:
- Понимать базовый синтаксис C#
- Работать с переменными и типами данных
- Использовать operators и выражения
- Применять условные конструкции и loops
- Работать с массивами и коллекциями
- Создавать и использовать методы

## 📚 Содержание

1. **[01_Variables.cs](01_Variables.cs)** - Variables и типы данных
2. **[02_Operators.cs](02_Operators.cs)** - Operators and Expressions
3. **[03_Conditionals.cs](03_Conditionals.cs)** - Conditional Statements
4. **[04_Loops.cs](04_Loops.cs)** - Loops
5. **[05_Collections.cs](05_Collections.cs)** - Arrays and Collections
6. **[06_Methods.cs](06_Methods.cs)** - Methods и параметры

## 🎮 Практические примеры

Все примеры в этом модуле связаны с игровой разработкой:
- Система подсчета очков
- Калькулятор урона
- Генератор случайных событий
- Базовая система инвентаря

## ✏️ Exercises

После изучения каждого файла выполните упражнения в конце файла. Решения находятся в папке `Exercises/`.

## ⏱️ Time изучения

Рекомендуемое время: **2-3 недели**
- 1-2 часа на каждый урок
- 2-3 часа на упражнения
